# HTML Authentication POC

This project contains simple HTML authentication pages created using only HTML.

No CSS or JavaScript is used. Only HTML structure and anchor tag redirections are implemented.

## Pages Included

1. login.html
   - Login page with links to Register, Forgot Password and Dashboard.

2. register.html
   - Registration page with link to Login page.

3. forgot-password.html
   - Forgot password page with link to Reset Password.

4. reset-password.html
   - Reset password page with link to Login page.

5. dashboard.html
   - Dashboard page with Logout link to Login page.

## Features

- Pure HTML pages
- Anchor tag navigation
- Proper redirections between pages

# Authentication System Styled

## Project Overview
This project is a styled authentication system built using HTML, Bootstrap 5, and custom CSS.  
The application includes multiple authentication-related pages designed with a responsive and professional UI.

## Features
- Login Page
- Registration Page
- Forgot Password Page
- Reset Password Page
- Dashboard Page

## Technologies Used
- HTML5
- Bootstrap 5
- Bootstrap Icons
- CSS3
- Google Fonts (Poppins)

## Custom Styling
The project includes custom CSS styling such as:
- Custom color scheme
- Google Fonts integration
- Hover effects on buttons and links
- Card box shadows
- Background gradient
- Smooth transitions
- Responsive design

## Responsive Design
The application is responsive and works on:
- Desktop (1920px and above)
- Laptop (1366px – 1920px)
- Tablet (768px – 1024px)
- Mobile (320px – 767px)
