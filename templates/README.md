# CampusPe Authentication System

## Project Description

This project is a simple **authentication system** built using **Python Flask for the backend** and **HTML, CSS for the frontend**.

It includes pages for **user registration, login, password reset, and dashboard access**.
The frontend templates are stored in the `templates` folder and styled using a custom CSS file.

---

## Project Structure

```
project/
│
├── app.py
│
└── templates/
    ├── dashboard.html
    ├── forgot.html
    ├── login.html
    ├── register.html
    ├── reset.html
    └── style.css
```

* User Login Page

* Custom CSS styling

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/Tippeshbh/campuspe_asses_front_end.git
```

2. Navigate to the project folder

```
cd campuspe_asses_front_end
```

3. Install Flask

```
pip install flask
```

4. Run the application

```
python app.py
```

5. Open the browser and go to

```
http://127.0.0.1:5000/
```
